## AI API Token计费模型

### 支持的模型和定价
- **GPT-4**: 输入 $0.03/1K tokens, 输出 $0.06/1K tokens
- **GPT-4-32K**: 输入 $0.06/1K tokens, 输出 $0.12/1K tokens
- **GPT-3.5-Turbo**: 输入 $0.0015/1K tokens, 输出 $0.002/1K tokens
- **Text-Embedding-Ada-002**: 输入 $0.0001/1K tokens, 输出 $0

- 详细使用日志
- 余额不足预警 (20%和10%阈值)
- 月度使用统计
- 每日使用量图表

  - JWT_SECRET - 使用强随机密钥

  - OPENAI_API_KEY - 替换为真实API Key
  - MONGODB_URI - 使用生产数据库地址
  - REDIS_URL - 使用生产Redis地址

### 数据库
- 需要启动MongoDB和Redis服务
- MongoDB: brew services start mongodb-community
- Redis: brew services start redis
