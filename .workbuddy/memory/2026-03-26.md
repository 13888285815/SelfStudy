## AI API Token计费模型

### 支持的模型和定价
- **GPT-4**: 输入 $0.03/1K tokens, 输出 $0.06/1K tokens
- **GPT-4-32K**: 输入 $0.06/1K tokens, 输出 $0.12/1K tokens
- **GPT-3.5-Turbo**: 输入 $0.0015/1K tokens, 输出 $0.002/1K tokens
- **Text-Embedding-Ada-002**: 输入 $0.0001/1K tokens, 输出 $0
